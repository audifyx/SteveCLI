# AGENT STEVE CLI

A command-line interface for Agent Steve, powered by OpenClaw.

## Features

- Full computer control
- File management
- Shell execution
- Browser automation
- And more!

## Setup

```bash
npm install -g agent-steve-cli
```

## Usage

```bash
steve <command>
```

## Development

```bash
git clone https://github.com/yourusername/agent-steve-cli.git
cd agent-steve-cli
npm install
```
